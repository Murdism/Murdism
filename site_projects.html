Perception systems, datasets, and prediction models built and tested on real autonomous platforms.

2023 — Present

End-to-End Multi-Modal Perception System

Full ROS2 perception pipeline deployed on real autonomous vehicles. Integrates YOLO 2D detection, PointPillars 3D LiDAR detection, camera-LiDAR fusion for 3D bounding boxes, multi-object tracking, and probabilistic trajectory prediction. Optimized via ONNX and TensorRT for real-time inference on NVIDIA hardware. Tested on EasyMile shuttles and the Pixmoving delivery robot in structured urban and closed-track environments.

2024 — Present · Co-First Author

EMT: Multi-Task Benchmark for Gulf-Region Driving

A 570,000+ annotation benchmark capturing underrepresented Arab Gulf driving conditions. I co-led annotation cleaning and validation, dataset tooling, cross-domain and in-dataset detection experiments, and multi-class multi-object tracking evaluation across several trackers. Built to measure how perception models generalize when deployment conditions differ from conventional benchmarks.

Guardian Safety Framework

Multi-layer autonomous safety stack combining geometric validation and VLM-based reasoning for consensus safety decisions. I implemented the camera-LiDAR fusion perception component. Under review at IEEE Access.

Custom 3D Detector via Camera-LiDAR Fusion

Designed a 3D detector that lifts YOLO 2D detections into 3D using camera-LiDAR fusion to address the limited pedestrian performance of standard LiDAR-only methods. Validated on ROS bags and in CARLA and Gazebo simulations.

Automated integration testing for Nav2 navigation stack using Gazebo and TurtleBot4. Batch testing protocols, regression metrics (success rate, execution time, path deviation), containerized with Docker for reproducible CI workflows.
